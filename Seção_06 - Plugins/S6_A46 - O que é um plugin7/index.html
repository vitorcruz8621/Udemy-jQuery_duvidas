O que é um plugin?

Um plugin é uma extensão do jQuery, assim como esse é uma extenção do JavaScript

Se jQuery é uma biblioteca de funções que resumem, em poucas linhas,
uma vasta gama de linhas de javascript para fazer um pequeno evento na webpage,
ou seja...

Se com JavaScript é uma linguagem de programação e foi usada para criar o jQuery...
com jQuery, eu posso criar um pequeno programa(plugin) para facilitar/agilizar a criação de webpages.
