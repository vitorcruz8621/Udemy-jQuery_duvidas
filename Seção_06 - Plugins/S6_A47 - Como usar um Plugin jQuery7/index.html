Como usar um Plugin jQuery?

Esse é o plugin jQuery Mask Plugin
Ele é um ótimo recurso para fazer a validação de campos(INPUT):
  • Datas e/ou Horas:minutos:segundos
  • Telefones (com/sem código operadora)
  • CPF
  • CNPJ
  • Endereço IP
  • Dinheiro
  • e muito mais...
